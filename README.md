# RecruitIQ – AI Resume Screening & Candidate Ranking System

RecruitIQ is a simple AI-powered resume screening tool designed to help HR teams and recruiters quickly analyze resumes and rank candidates based on job requirements.

The system extracts key information from uploaded resumes and evaluates them using keyword analysis to determine how well a candidate matches a specific role.

---

## Features

* Upload resumes for analysis
* Extract important information from resumes
* Keyword-based candidate evaluation
* Candidate scoring system
* Simple and user-friendly interface
* Backend processing using Python

## How It Works

1. The user uploads a resume file.
2. The backend processes the resume.
3. Keywords related to job skills are analyzed.
4. The system calculates a score based on keyword matches.
5. The result is displayed to the user.
